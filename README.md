# 🏠 Bengaluru Home Price Prediction System

This project is a Machine Learning project developed during the *DecodeLabs Data Science Internship* to predict home prices in Bengaluru using real estate market data and predictive modeling techniques.

---

## 🚀 Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

---

## 📂 Project Files

- bengaluru_home_price_prediction.ipynb
- Bengaluru_House_Data.csv
- README.md
- requirements.txt

---

## 🎯 Project Objective

The main objective of this project is to build a Machine Learning model capable of estimating property prices in Bengaluru based on different housing features such as:

- Location
- Area (Square Feet)
- Number of Bedrooms (BHK)
- Bathrooms
- Property Size
